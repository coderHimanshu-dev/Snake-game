# 🐍 Snake Game

A classic Snake Game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Smooth controls
- Score tracking
- Game over detection
- Responsive design

## 🎮 How to Play
- Use arrow keys to move the snake
- Eat food to grow
- Avoid hitting walls or yourself

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
